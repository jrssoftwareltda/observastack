# Changelog

## 0.1.0 - 2026-07-09

Initial ObservaStack case implementation.

### Added

- Custom `/health` endpoint.
- Spring Boot Actuator health, info, metrics and Prometheus endpoints.
- Micrometer metrics for observed API traffic, errors, latency and active alerts.
- JSON structured logging with Logstash Logback Encoder.
- Traffic simulator endpoint.
- Alert simulator endpoint.
- Before/after optimization comparison endpoint.
- Static HTML dashboard.
- Prometheus and Grafana Docker Compose stack.
- GitHub Actions CI with Maven build, tests, Docker build and Trivy scans.
- Dependabot configuration.
- Website case metadata for JRS Software.
